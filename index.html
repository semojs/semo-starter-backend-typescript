封装各种公司底层基础设施，不需要考虑怎么连接，拿过来就可以使用。

内置 Semo 框架

通过集成 Semo，使得项目可以获得 Semo 的功能和扩展性。

# 借助 Semo 命令行工具，让整个开发流程更加顺畅，semo create 命令会帮助下载，yarn install 和 重置 .git 目录
npm i -g @semo/cli
semo new YOUR_PROJECT REPL_URL
cd YOUR_PROJECT
yarn watch # 用于实时编译 typescript
yarn start # 启动开发运行环境
